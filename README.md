# CHATBOT_JIRA_VERMEG
Voici une version professionnelle de moins de **350 caractères** :  > Projet réalisé dans le cadre d’un stage de 2 mois chez Vermeg. Développement d’un chatbot intelligent basé sur l’IA permettant de rechercher et analyser des tickets Jira en langage naturel. Technologies utilisées : Angular, Spring Boot, Jira REST API, JQL et IA (LLM).
